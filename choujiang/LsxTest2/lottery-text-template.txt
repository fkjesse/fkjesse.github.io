# 年会抽奖文本格式模板

## -1. 技术应用说明

### 数据处理
- Papa Parse：CSV解析
- xlsx.js：Excel处理
- iconv-lite：编码转换

### 数据验证
- Joi：数据验证
- validator.js：格式检查

### 文件处理
- fs-extra：文件操作
- chokidar：文件监听
- archiver：文件压缩

## 1. 基本格式要求
- 每行一个参与者信息
- 信息之间用制表符(Tab)或逗号分隔
- 不要包含特殊字符和emoji表情
- 编码格式使用UTF-8

技术应用：
- 编码处理：
  - iconv-lite：编码转换
  - Buffer：二进制处理
- 文本处理：
  - String API
  - RegExp：格式验证

## 2. 标准格式示例
工号    姓名    部门    职位
001     张三    技术部  工程师
002     李四    市场部  经理
003     王五    财务部  主管

## 3. 数据字段说明
必填字段：
- 工号：唯一标识符，纯数字，不少于3位
- 姓名：2-4个汉字
- 部门：部门全称

可选字段：
- 职位
- 入职时间
- 联系方式

## 4. 注意事项
1. 请检查是否有重复工号
2. 确保所有必填信息完整
3. 删除离职员工信息
4. 更新部门名称为最新组织架构
5. 去除多余的空格和换行

## 5. 文件命名规范
格式：公司名-年会抽奖-年份.txt
示例：XX科技-年会抽奖-2024.txt

## 6. 数据示例（含可选字段）

## 7. 数据验证清单
□ 检查工号唯一性
□ 验证必填字段完整性
□ 确认部门名称准确性
□ 清理特殊字符
□ 统一格式和编码
